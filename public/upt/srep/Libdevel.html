  1. DOUBLE-BUFFER (DBE), the Double Buffer extension;
  2. DPMS, the VESA Display Power Management System extension;
  3. Extended-Visual-Information (EVI), an extension for gathering extra information about the X server's visuals;
  4. LBX, the Low Bandwidth X extension;
  5. MIT-SHM, the MIT X client/server shared memory extension;
  6. MIT-SUNDRY-NONSTANDARD, a miscellaneous extension by MIT;
  7. Multi-Buffering, the multi-buffering and stereo display extension;
  8. SECURITY, the X security extension;
  9. SHAPE, the non-rectangular shaped window extension;
  10. SYNC, the X synchronization extension;
  11. TOG-CUP, the Open Group's Colormap Utilization extension;
  12. XC-APPGROUP, the X Consortium's Application Group extension;
  13. XC-MISC, the X Consortium's resource ID querying extension;
  14. XTEST, the X test extension (this is one of two client-side implementations; the other is in the libXtst library, provided by the libxtst6 package);

libXext also provides a small set of utility functions to aid authors of client APIs for X protocol extensions.
