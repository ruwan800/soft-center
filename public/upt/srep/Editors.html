Descriptionsmall, friendly text editor inspired by Pico

GNU nano is an easy-to-use text editor originally designed as a replacement for Pico, the ncurses-based editor from the non-free mailer package Pine (itself now available under the Apache License as Alpine).

However, nano also implements many features missing in pico, including:

  1. feature toggles;
  2. interactive search and replace (with regular expression support);
  3. go to line (and column) command;
  4. auto-indentation and color syntax-highlighting;
  5. filename tab-completion and support for multiple buffers;
  6. full internationalization support.
